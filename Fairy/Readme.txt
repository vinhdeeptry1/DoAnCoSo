1) Mở folder trên Visual Studio Code.

2) Cài đặt môi trường ảo
python -m venv venv
.\venv\Scripts\activate

3) Tải thư viện
pip install google-generativeai
pip install pyttsx3
pip install SpeechRecognition

4) Chạy chương trình:
Chọn file UI.py -> F5 hoặc Ctrl F5 hoặc nút chạy chương trình Python.

5) Cơ sở dữ liệu:
Cơ sở dữ liệu được tạo bằng file JSON nên có thể dùng cơ sở dữ liệu đã có sẵn, hoặc tạo mới bằng cách đăng ký tài khoản mới.
